# Application de Gestion de Ventes

## Description

Application desktop pour enregistrer et gérer les ventes d'un commerce.

## Installation

Double-cliquez sur `GestionVentes.exe`

## Utilisation

### Enregistrer une vente

1. Ouvrez l'onglet "Enregistrer une vente"
2. Remplissez les champs :
   - Nom du produit
   - Quantité
   - Prix unitaire en F CFA
3. Cliquez sur "Enregistrer la vente"
4. Le prix total est calculé automatiquement

### Consulter l'historique

- Toutes les ventes sont affichées dans le tableau
- Les statistiques (total et nombre de ventes) sont mises à jour automatiquement

### Supprimer une vente

1. Sélectionnez une vente dans le tableau
2. Cliquez sur l'icône de suppression (symbole "X") associée à la vente sélectionnée
3. Confirmez la suppression

### À propos

Consultez l'onglet "À propos" pour les informations sur l'application.

## Données

Les ventes sont stockées dans un fichier `ventes.db` créé automatiquement.
